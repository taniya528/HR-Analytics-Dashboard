# HR-Analytics-Dashboard

## 🔹Overview

HR Analytics Dashboard built in Power BI to analyze employee performance, attrition, salary trends, department-wise insights, gender distribution, and experience analysis. This project demonstrates data visualization, KPI tracking, interactive filtering, and business intelligence reporting using Power BI and Excel.

Visualized gender-wise employee distribution.

Tracked department-wise attrition trends.

Analyzed employee experience groups and workforce distribution.
